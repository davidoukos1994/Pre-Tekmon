# Μετρήσεις pre-TEKMON

Στατική, responsive εφαρμογή για καταχώρηση μετρήσεων παραμέτρων παραγωγής.

## Δυνατότητες

- Αυτόματη εμφάνιση τρέχουσας ημερομηνίας και ώρας.
- Επεξεργάσιμα πεδία ανά τομέα μετρήσεων.
- Αποθήκευση καταχωρήσεων τοπικά στον browser (`localStorage`).
- Ιστορικό με άνοιγμα, επεξεργασία, εκτύπωση και διαγραφή καταχώρησης.
- Εξαγωγή ιστορικού σε JSON και CSV.
- Εισαγωγή αντιγράφου ασφαλείας JSON.
- Φωτεινό/σκοτεινό θέμα.
- Λειτουργία ως PWA και offline μετά την πρώτη φόρτωση.

## Ανέβασμα στο GitHub Pages

1. Δημιούργησε νέο repository με όνομα `metriseis-pre-tekmon`.
2. Ανέβασε όλα τα αρχεία αυτού του φακέλου στη ρίζα του repository.
3. Πήγαινε **Settings → Pages**.
4. Στο **Build and deployment**, επίλεξε **Deploy from a branch**.
5. Επίλεξε branch `main` και folder `/ (root)` και πάτησε **Save**.
6. Μετά από λίγα λεπτά θα εμφανιστεί το URL της εφαρμογής.

## Σημαντικό για τα δεδομένα

Οι καταχωρήσεις αποθηκεύονται μόνο στη συγκεκριμένη συσκευή και στον συγκεκριμένο browser. Για αντίγραφο ασφαλείας χρησιμοποίησε τα κουμπιά **Εξαγωγή JSON** ή **Εξαγωγή CSV**.

Για κοινό ιστορικό σε πολλές συσκευές χρειάζεται backend/database (π.χ. Firebase ή Supabase).
